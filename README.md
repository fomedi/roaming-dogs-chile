# Free-Roaming Dog Density — Chile 2021

**Estimating free-roaming dog density to support dog-bite prevention and population management in Chile**

Mardones, F.O. · Fernández, M. · Benavides, J. · Atero, N. · Diethelm-Varela, B. · Córdova, F. · Salgado-Caxito, M.

---

## Overview

This repository contains the complete, reproducible analysis for the manuscript above. Georeferenced transect observations from the First National Study on Responsible Companion Animal Ownership (PTRAC–SUBDERE, December 2021) are used to estimate roaming-dog density across 25 Chilean districts, model temporal structure, screen district-level correlates, model encounter probability, and characterise spatial clustering along transects.

**Key findings:**
- Pooled density: **13.7 dogs/km** (95% CI 13.1–14.3) across 25 districts; 14-fold variation among well-sampled districts (3.0–42.0 dogs/km)
- Between-district variance dominates (ICC = 0.52); morning peak ~18 dogs/km at 09:00
- None of 27 district-level predictors survives FDR correction for density outcome
- **Neutering rate** is the only significant predictor of encounter probability (OR ≈ 0.54, p = 0.035): each SD increase in district sterilisation coverage halves the odds of observing a roaming dog
- No association between roaming density and dog-bite incidence
- Dogs distributed approximately randomly within districts (median aggregation ratio ≈ 1.0)

## Reproducing the analysis

### Prerequisites

R (≥ 4.2) with the following packages:

```r
install.packages(c(
  "readxl", "writexl", "tidyverse", "janitor", "lubridate",
  "geosphere", "splines", "gridExtra",
  "glmmTMB", "car", "DHARMa", "performance",
  "broom", "broom.mixed", "emmeans", "MASS",
  "ggcorrplot", "FactoMineR", "factoextra", "ggrepel",
  "openxlsx"
))
```

### Running the script

1. Place `roaming.xlsx` and `bites_2019_24.xlsx` in your working directory  
   *(Note: the bite file is not shared publicly due to data governance; contact the corresponding author)*
2. Open `R/Roaming_paper_complete.R` and set `data_dir` at the top
3. Run end-to-end — the script creates `data/`, `tables/`, and `figures/` subdirectories automatically

The script is self-contained and runs in a single session (~5–10 minutes).

## Data notes

### Input: `roaming.xlsx`

Two sheets used:
- **`original`** — Epicollect5 transect records (n = 1,986 raw; n = 1,541 after filtering). Columns: GPS coordinates, timestamp, district, dog count.
- **`survey`** — District-level attributes (25 rows × 80 columns) including owned-dog demographics, socioeconomic indices (IDC2, IDH 2005), infrastructure, and ownership behaviour proportions (Salgado-Caxito et al., 2023).

### Key variable decisions (verified against current sheet)

| Column used | Column excluded | Reason |
|---|---|---|
| `densidad_humana_km2` | `densidad` | Different sources (r = 0.69, ratio varies 0.8–140×) |
| `area km2` | `Superficie` | Major discrepancies for several districts (max diff 8,796 km²) |
| `vets` | `n_vets` | Identical (r = 1.00) |
| `hogares` | `n_hogares` | Near-identical (r = 0.9998) |
| `IDC2` | `IDC` | Identical (r = 1.00) |
| `neutered` | `neutering` | `neutered` is a proportion (0.24–0.59); `neutering` is old binary |
| *(excluded)* | `total_mascotas`, `densidad_mascotas_km2` | Dogs + cats combined (~1.44× dogs); not dog-specific |
| *(excluded)* | all `gatos_*` columns | Cat columns deleted from current sheet |

### Transect data filtering

- Time window: 08:00–19:00, excluding 12:00–13:59
- GPS relocation exclusion: consecutive steps > 500 m
- Low-effort threshold: < 1 km walked (Putaendo, Mulchén, Viña del Mar)
- Excluded districts: La Estrella (insufficient records), Hualqui and Santa María (single point), Putre (no records in window)

## Results summary

| Outcome | n | Key result |
|---|---|---|
| Pooled density | 25 districts | 13.7 dogs/km (95% CI 13.1–14.3) |
| Variation | 22 well-sampled | 3.0–42.0 dogs/km (14-fold) |
| Between-district variance | — | ICC = 0.52 |
| Diurnal pattern | — | Peak 09:00 (~18 dogs/km); −40% by 14:00–16:00 |
| Demographic correlates | 25 predictors | All null (Table 3) |
| Extended correlates | 27 predictors | 4 nominally significant; none survives FDR |
| Multivariate NB | IDC2 + neutered + always_inside | All null; LRT p = 0.073 |
| **Encounter rate model** | Quasi-binomial | **Neutering: OR = 0.54 (0.31–0.91), p = 0.035** |
| Spatial clustering | 25 districts | Median AR ≈ 1.0 (approximately random) |
| Bite–density link | 8 scenarios | All null |
